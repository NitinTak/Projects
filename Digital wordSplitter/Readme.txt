
Project was developed on visual studio 2015.

Project: WordSplitter
Does not require any external input. Will find all possible unique letter combinations with length greater than or equal to 3 for the word "digital".
Output will be a list of these words which will be saved in the current working directory of the project with the file name: "SD_digital_WordSplitter.txt"

Displays the following message on console:
"6975 total possible unique letter combinations with length >= 3 exist for the word 'digital'.
Writing to file (Filename: SD_digital_WordSplitter.txt) in current working project directory...
File write completed."
