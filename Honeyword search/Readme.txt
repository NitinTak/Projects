Quantcast: Honeycomb Word Search

Challenge was coded and developed as a console application on visual studio 2015. 
To run:
- Open a new default console application in Visual Studio 2015.
- Replace "Program.cs" in a default application by the file attached.
- Provide the following 2 command line arguments required:
1. Path of honeycomb input text file.
2. Path of dictionary input text file.
- Run the program.

Output will be displayed on the console and will have the valid strings found in honeycomb structure sorted in alphabetical order.
