
Project was developed on visual studio 2015.

Project: Maze
Requires a file containing a maze to be passed as command line arguments.
Output will be the coordinates of path to be followed from Start to Finish inclusive in the maze when viewed in 2-dimension.
Displays appropriate messages to user on console wherever applicable.
For example: if the given maze is:
1S111
10001
11101
10001
11F11

Then the program would present the path as:
(0,1)
(1,1)
(1,2)
(1,3)
(2,3)
(3,3)
(3,2)
(4,2)
